# E-Commerce-Company-SQL-based-Case-Study
A real-world SQL case study on an e-commerce company, using MySQL to analyze customer distribution and answer business-driven questions.

# Project Overview :
This project focuses on analyzing an E-Commerce Company database using SQL to extract meaningful business insights. The case study simulates real-world data analyst tasks such as understanding table structures, performing aggregations, filtering data, and answering business-driven questions using SQL queries.
The SQL file contains multiple analytical queries designed to explore customer distribution, order patterns, and location-based insights.\

# Tools & Technologies
SQL (MySQL)
MySQL Workbench

# Business Questions Answered
  How many customers are present in each location?
  What is the overall customer distribution?
  How can SQL be used to summarize business data effectively?
  Each query is written with a business-oriented approach rather than raw technical output. etc etc
